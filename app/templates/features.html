Feature Description


Duration

Duration: Length of time duration of the connection

Type: Continuous
Value Type: Integers
Range: 0 - 42862


Protocol Type

Protocol_type: Protocol used in the connection

Type: Categorical
Value Type: String


Service

Service: Destination networ service used

Type: Categorical
Value Type: String


Flag

Flag: Status of the connection - Normal or Error

Type: Categorical
Value Type: String


Src Bytes

Src_bytes: Number of data bytes transferred from source to destination in single connection

Type: Continuous
Value Type: Integers
Range: 0 - 381709090


Dst Bytes

Dst_bytes: Number of data bytes transferred from source to destination in single connection

Type: Continuous
Value Type: Integers
Range: 0 - 5151385


Land

Land: If source and destination IP addresses and port numbers are equal then, this variable takes value 1 else 0

Type: Binary
Value Type: Integers


Wrong Fragment

Wrong_fragment: Total number of wrong fragments in this connection

Type: Discrete
Value Type: Integers
Range: 0 - 3


Urgent

Urgent: Number of urgent packtes in this connection. Urgent packets are packets with the urgent bit activated

Type: Discrete
Value Type: Integers


Hot

Hot: Number of "hot" indicators inn this content such as: entering a system directory, creating programs and executing programs

Type: Continuous
Value Type: Integers
Range: 0 - 101


Num Failed Logins

Num_failed_logins: Count of failed login attempts

Type: Continuous
Value Type: Integers
Range: 0 - 4


Logged In

Logged_in: Login Status : 1 if successfully logged in else 0

Type: Binary
Value Type: Integers


Num Compromised

Num_compromised: Number of "compromised" conditions

Type: Continous
Value Type: Integers
Range: 7479


Root Shell

Root_shell: 1 if root shell is obtained else 0

Type: Binary
Value Type: Integers


Su Attempted

Su_attempted: 1 if "su root" command attempted or used else 0

Type: Discrete (Dataset contains '2' value
Value Type: Integers
Range: 0 - 2


Num Root

Num_root: Number of "root" accesses or number of operatins performed as a root in the connection

Type: Continuous
Value Type: Integers
Range: 0 - 7468


Num File Creations

Num_file_creations: Number of file creation operations in the connection

Type: Continuous
Value Type: Integers
Range: 0 - 100


Num Shells

Num_shells: Number of shell prompts

Type: Continuous
Value Type: Integers
Range: 0 - 2


Num Access Files

Num_access_files: Number of operations on access control files

Type: Continuous
Value Type: Integers
Range: 0 - 9


Num Outbound Cmds

Num_outbound_cmds: Number of outbound commands in an ftp session

Type: Continuous
Value Type: Integers
Range: 0


Is hot Logins

Is_hot_login: 1 if the login belongs to the "hot" list i.e., root or admin else 0

Type: Binary
Value Type: Integers
Range: 0 - 1


Is Guest Login

Is_guest_login: 1 if the login is a "guest" login; else 0

Type: Binary
Value Type: Integers
Range: 0 - 1


Count

Count: Number of connections to the same destination host as the current connection in the past two seconds

Type: Discrete
Value Type: Integers
Range: 0 - 511


Srv Count

Srv_count: Number of connections to the same service (port number) as the current connection in the past two seconds

Type: Discrete
Value Type: Integers
Range: 0 - 511


Serror Rate

Serror_rate: The percentage of connections that have activated the flag (4) s0, s1, s2 or s3, among the connections aggregated in count(23)

Type: Discrete
Value Type: Floats (hundredths of decimal
Range: 0 - 1


Srv Serror Rate

Srv_serror_rate: The percentage of connections that have activated the flag (4) s0, s1, s2 or s3, among the connections aggregated in srv_count (24)

Type: Discrete
Value Type: Floats (hundredths of decimal
Range: 0 - 1


Rerror Rate

Rerror_rate: The percentage of connections that have activated the flag (4) REJ, among the connections aggregated in count (23)

Type: Discrete
Value Type: Floats (hundredths of decimal
Range: 0 - 1


Srv Rerror Rate

Srv_rerror_rate: The percentage of connections that have activated the flag (4) REJ, among the connections aggregated in srv_count (24)

Type: Discrete
Value Type: Floats (hundredths of decimal
Range: 0 - 1


Same Srv Rate

Same_srv_rate: The percentage of connections that were to the same service, among the connections aggregated in count (23)

Type: Discrete
Value Type: Floats (hundredths of decimal
Range: 0 - 1


Diff Srv Rate

Diff_srv_rate: The percentage of connections that were to different services, among the connections aggregated in count (23)

Type: Discrete
Value Type: Floats (hundredths of decimal
Range: 0 - 1


Srv Diff Host Rate

Srv_diff_host_rate: The percentage of connections that were to different destination machines among the connections aggregated in srv_count (24)

Type: Discrete
Value Type: Floats (hundredths of decimal
Range: 0 - 1


Dst Host Count

Dst_host_count: Number of connections having the same destination host IP address

Type: Discrete
Value Type: Integers
Range: 0 - 255


Dst Host Srv Count

Dst_host_srv_count: Number of connections having the same port number

Type: Discrete
Value Type: Integers
Range: 0 - 255

Type: Discrete
Value Type: Floats (hundredths of a decimal)
